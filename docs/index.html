コンソールで変数p1,p2の値を変えれば曲線をいじれるよ。例:
p1={x:0.2,y:0};
p2={x:0.8,y:1};

★超簡単な説明★

ベジェ曲線は曲線を書くのには便利だけど、実はイージングとして使うのはけっこう難しい。だってベジェ曲線の関数はf(t)=座標だけど、イージングはf(x)でyの座標を求めないといけないからね。
こんな中で直接yの値を計算する方法はあるかもしれないけど絶対難しい。だから、直接求めるんじゃなくて近い値を出すようにしている。
この方法は使ってないけど一番分かりやすい方法は、まず100個ぐらい値を出して、そこから一番近い値を選ぶ方法。でもそれじゃああんま正確じゃないし、重い。もっと効率的にしたい。そこで使うのは2進数。
まずtに0と1を入れて計算して、どっちが近いか見る。もし0の方が近かったら、次はは0と0.5、1の方が近かったら次は0.5と1を比べて...0.25,0.125,0.0625,,,,っていうことをずっと繰り返すとかなり近い値を速く計算できる。
なんとなくわかったかな?このイージングはこの方法を使って近い値を出してるよ。ソースコードはGithubにあるからもっと詳しく知りたかったら見てみてね。
